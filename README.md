# MailParser

Утилита для разбора EML файлов и извлечения из них текстового и HTML контента.

> EML файлы — это стандартный формат файлов электронной почты, который используется для хранения сообщений электронной почты. Они содержат заголовки, тело сообщения и вложения.


Программа парсит файлы и сохраняет результат в JSON формате, который будет использован скриптом на Python для дальнейшей конвертации текста в .DOCX файл по шаблону


На данный момент программа может:
- Читать и выводить на экран содержимое EML файлов (в расшифрованном виде);
> Пример:
```text
Файл: D:\Code\MailParser\input\eml\Test.eml
Отправитель: <src@gmail.com>
Получатель:  <dst@gmail.com>
Сообщение:
[18.09.2019, 10:26] Привет Golang! Давно я не вникал в твой синтаксис
```
